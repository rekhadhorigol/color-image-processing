### Color Image Processing
 
Explores **color image transformations** using matrix operations on RGB channels, with analysis of invertibility.
 
**Operations Covered:**
- Upload and display a color image
- Separate and display individual **R, G, B channels**
- Convert to **grayscale** using luminance weights
- Apply **sepia transformation** using a 3×3 matrix
- Isolate the **red channel**
- **Permute channels** - RGB -> BGR
- **Remove blue channel**
- **Invert** color image
- Custom **saturation matrix** transformation
- **User-defined color adjustment** matrix
- **Gamma correction** (γ = 1.1)
- Discussion of invertibility — which transformations can be reversed and why
 
**Tools & Libraries:** numpy, matplotlib, PIL
